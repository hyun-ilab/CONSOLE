# Library

공부자료, 디자인 원칙, Codex/LLM 읽기자료, 재사용 프롬프트를 모으는 공간입니다.

## Folders

- `inbox/`: 아직 성격이 확정되지 않은 자료.
- `study/`: 강의, 논문, 책, 개인 학습 노트.
- `project_context/`: 프로젝트 목표, 결정 사항, 현재 상태를 설명하는 참고자료.
- `design_principles/`: 디자인 원칙, 레퍼런스, 스타일 기준.
- `agent_prompts/`: 재사용 프롬프트, 작업 지시문, 체크리스트.

## Rule

작업자가 반드시 지켜야 하는 규칙은 루트 `AGENTS.md`에 둡니다. 참고하면 좋은 배경자료만 이 폴더에 둡니다.
