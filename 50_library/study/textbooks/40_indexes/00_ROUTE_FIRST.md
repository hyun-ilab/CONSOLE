## Source Rules

- PDF sources are cited by PDF page marker, for example `PDF page N`.
- EPUB sources are cited by EPUB spine marker plus section heading.
- `20_converted_md/` is a page-order readable copy for PDF reading, not a summary.
- `PROMPT_TEMPLATES.md` is only reusable prompt text; do not use it as answer evidence.
- Important claims must be rechecked in `15_source_text/` or, for PDFs, the matching `20_converted_md/` readable copy.
- Avoid reading both `15_source_text/` and `20_converted_md/` end to end for the same PDF unless extraction quality or line context needs cross-checking.
